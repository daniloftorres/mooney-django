## Notação Big O

        - Big O notation
        Classificação da complexidade dos algoritimos
        [Analise Sintótica, entender isso]

        Expressão padrão : O(n)
        O : Custo/Tempo de processamento
        n : Variavel (entrada ou volume de elementos)
